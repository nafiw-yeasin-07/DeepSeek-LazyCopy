# DeepSeek LazyCopy

Copy full DeepSeek responses instantly with one click. No highlighting. No hassle.

## Features

- One-click copy for entire responses
- Clean copied output
- Lightweight and fast
- No external services

## Installation (Chrome)

1. Download the latest ZIP from Releases
2. Extract the ZIP
3. Open chrome://extensions
4. Enable Developer Mode (top right)
5. Click "Load unpacked"
6. Select the folder containing manifest.json

## Usage

Open DeepSeek.
Click the Copy button added by the extension.
The full response is copied to your clipboard.
